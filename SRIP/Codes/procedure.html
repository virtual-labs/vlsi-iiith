SCHEMATIC DESIGN OF D-LATCH AND D FLIP FLOP



PROCEDURE



D LATCH

COMPONENTS REQUIRED

  • 2*1 Multiplexer
  • 2 inverters
  • input pin
  • output pin
  • clock

CONNECTING THE COMPONENTS

  • connect output port of2*1 multiplexer to the input of the 1st inverter.
  • Now connect output of 1st inverter to output pin and input of 2nd inverter.
  • connect input pin to the 1st port of the multiplexer.
  • connect output of2nd invrter to the second input line of multiplexer.
  • connect clock to multiplexer

OBSERVATION AND SIMULATIONS

  • press the simulation button on the top.
  • observe the graph and verify that it works as pass transistor

POSITIVE EDGE FLIP FLOP

COMPONENTS REQUIRED

  • 2 I/O latch
  • input pin
  • output pin
  • clock

CONNECTING THE COMPONENTS

  • connect the input port of the 1st latch with input port of 2nd latch.
  • connect input pin to input port of 1st latch
  • connect output pin to output port of 2nd latch
  • connect clock to 1st latch
  • connect clock bar to 2nd latch

OBSERVATION AND SIMULATIONS

  • press simulation button on the top
  • observe the graph and verify that it works as positive level d flip flop
